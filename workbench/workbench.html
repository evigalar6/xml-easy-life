XPath generation uses:
1. Well-formed XML
2. Current cursor position
3. Open tag stack before the cursor

If your cursor is on indentation/whitespace inside an element, it usually still resolves correctly.

Ambiguity can happen:
- between sibling elements
- before root / after document end
- when XML is invalid

In ambiguous cases, the generator may fall back to a nearby/root element.
